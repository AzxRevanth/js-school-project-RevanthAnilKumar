# Timeline App - Styles

## CSS Styling Overview

This stage adds a responsive, polished layout to the Timeline App using CSS.

### Key features:
- Responsive timeline layout using CSS Grid for arranging event items.
- Styled header with typography and color scheme.
- Simple placeholder styles for event markers (colored left border as dot).
- Modal base styles included (hidden by default).
- Responsive breakpoints for mobile (<768px), tablet (768–1023px), and desktop (≥1024px).

### How to test
- Open the app in different screen sizes and verify timeline items rearrange correctly.
- Check header styling and button appearance.
- Confirm modal is hidden by default and styled properly when shown.
