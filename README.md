# Task 3 - Javascript

## Goal
An interactive timeline that showcases key moments in Ferrari’s history, using vanilla JavaScript for dynamic event loading and modal-based details.

## Features
- **Data-driven content**: `data/events.json` contains 8 Ferrari-related events (fields: `year`, `title`, `description`, `imageURL`, `category`).
- **Dynamic rendering**: JavaScript fetches and displays events in `#timeline`.
- **Interactive modals**: Click an event marker to view more information.
- **Modal close functionality**: Easily close via button or overlay click.

## How to Run
1. Clone this repository.
2. Open `index.html` in your browser.
3. Interact with the timeline — click any marker to see Ferrari event details.

## About the Data
The `events.json` file contains historical and iconic Ferrari moments, such as the launch of legendary models, key racing victories, and major milestones in Ferrari’s automotive legacy.

## How to Test
1. Ensure all events load correctly from `events.json`.
2. Confirm clicking markers opens the modal with accurate Ferrari history.
3. Verify modal closes properly.
