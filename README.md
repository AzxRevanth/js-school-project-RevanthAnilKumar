# Timeline React App

## Goal
Rebuild the timeline UI as a React application for better modularity and easier future enhancements.

## Features
- Bootstrapped with Vite (React + TypeScript template)
- Components:
  - Header with logo and theme toggle
  - Timeline that maps over event data
  - EventMarker for each year/title entry
  - EventModal for detailed event view
  - Placeholder for optional FilterPanel
- State management with React hooks (useState, useEffect)
- Loads data dynamically from `events.json`
- Styling with `App.css`
